# Tic Tac Toe

This is a basic implementation of the game Tic Tac Toe in JavaScript.

## Live Demo

https://code.lucaseverett.dev/tic-tac-toe/

## Getting Started

Clone this repo:

```sh
git clone https://github.com/lucaseverett/tic-tac-toe.git
```

Install NPM dependencies:

```sh
npm install
```

Start dev server:

```sh
npm start
```
